## **Types of Business Models**


| **Category**            | **Business Model**             | **Brief Description**                                                   |
|-------------------------|--------------------------------|-------------------------------------------------------------------------|
| **Revenue Generation**  | Subscription Model             | Recurring revenue from regular payments (e.g., Netflix, Spotify)        |
| **Revenue Generation**  | Freemium Model                 | Basic service free, premium features paid (e.g., Dropbox, LinkedIn)     |
| **Revenue Generation**  | Pay-per-Use / Usage-Based      | Charge based on consumption (e.g., AWS, Uber)                           |
| **Revenue Generation**  | One-Time Purchase              | Single payment for ownership (e.g., retail products, software licenses) |
| **Revenue Generation**  | Advertising Model              | Free service funded by ads (e.g., Google, Facebook, YouTube)            |
| **Revenue Generation**  | Commission / Marketplace Model | Earn commission on transactions (e.g., Amazon, Airbnb, Uber)            |
| **Revenue Generation**  | Licensing Model                | Grant rights to use IP for a fee (e.g., software, patents)              |
| **Revenue Generation**  | Franchise Model                | Sell rights to operate under brand (e.g., McDonald’s, Subway)           |
| **Product/Service**     | Product-Based                  | Sell physical or digital goods (e.g., Apple, Nike)                      |
| **Product/Service**     | Service-Based                  | Sell expertise/time (e.g., consulting, legal, plumbing)                 |
| **Product/Service**     | SaaS (Software as a Service)   | Cloud-based software on subscription (e.g., Salesforce, Zoom)           |
| **Product/Service**     | PaaS / IaaS                    | Platform or Infrastructure as a Service (e.g., AWS, Heroku)             |
| **Distribution**        | Direct-to-Consumer (D2C)       | Sell directly to customers (bypassing retailers)                        |
| **Distribution**        | B2B (Business-to-Business)     | Sell to other businesses                                                |
| **Distribution**        | B2C (Business-to-Consumer)     | Sell directly to individual consumers                                   |
| **Distribution**        | B2G (Business-to-Government)   | Sell to government agencies                                             |
| **E-commerce Specific** | Dropshipping                   | Sell products without holding inventory                                 |
| **E-commerce Specific** | Print-on-Demand                | Custom products printed when ordered                                    |
| **E-commerce Specific** | Affiliate Marketing            | Earn commission by promoting others’ products                           |
| **Innovative/Hybrid**   | Razor-Blade Model              | Sell cheap durable item, profit on consumables (e.g., Gillette)         |
| **Innovative/Hybrid**   | Bundling Model                 | Sell products/services together at a discount                           |
| **Innovative/Hybrid**   | Crowdfunding                   | Raise funds from many people (e.g., Kickstarter)                        |
| **Innovative/Hybrid**   | Data Monetization              | Sell or use customer data for revenue                                   |
| **Innovative/Hybrid**   | Platform Ecosystem             | Create a platform where others build value (e.g., Apple App Store)      |
| **Social/Impact**       | Social Enterprise              | Profit with social/environmental mission                                |
| **Social/Impact**       | Nonprofit with Earned Income   | Revenue-generating activities to support mission                        |

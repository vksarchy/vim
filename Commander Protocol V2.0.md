**[ENGINEERING LENS — Applying Musk 5-Step as a System]**

1. **Define the System** — The system is: how Sid processes tasks on a Monday. Inputs: task list, calendar, backlog. Output: executed work. Constraint: solo operator, two businesses, limited cognitive bandwidth.

2. **First Principles** — Most tasks exist because they always existed. Most processes are inherited, not designed. The question is never "how do I do this faster" — it's "should this exist at all."

3. **Failure Modes** — I skip Step 1 (questioning requirements) because it feels slow. I jump to Step 3 (simplify) without doing Step 2 (delete). Result: I optimize garbage instead of eliminating it.

4. **Min Viable Intervention** — Before touching any task, run a 30-second filter: *Does this task survive Step 1 and 2?* Only then work on it.

5. **Feedback Loop** — End of each day, 5-minute review: which tasks did I delete today? Which did I just execute on autopilot? That ratio is the signal.

6. **Iterate** — Week 1: conscious application. Week 3: semi-automatic. Week 6: it's just how I think.

**[STEP 3 — MISSION + 3 STEPS + FUCKUP FIX]**

**Mission:** Run today's Monday task list through all 5 Musk steps explicitly, once, so I know what the filter feels like in practice.

**Step 1 — Morning (before starting anything):** Pull out your task list. For each item, ask one question: *"Who decided this needs to happen, and were they right?"* Cross out anything that survives only because of inertia.

**Step 2 — Before each task block:** Ask: *"If I don't do this subtask, does the outcome break?"* Delete the ones that don't. What's left is your skeleton.

**Step 3 — End of day, 10 minutes:** Look at what you actually did. Identify one thing that could be templated, delegated, or scripted (InLingua follow-ups, Kappini lead qualification — both are candidates). That's your automation queue.

**Fuckup Fix:** If I fall into execution mode and forget to run the filter — I catch it at end-of-day review, not mid-task. The daily review is the safety net. If I skip the review too, the week is lost. So the review is non-negotiable. 10 minutes. Calendar block it now.
